## 🚀 Cara Deploy ke GitHub Pages

### Langkah 1 — Buat Akun GitHub
Jika belum punya, daftar di [github.com](https://github.com)

### Langkah 2 — Buat Repository Baru
1. Klik tombol **"New"** atau buka [github.com/new](https://github.com/new)
2. Isi **Repository name** dengan: `nikah-yukk`
3. Pilih **Public**
4. Klik **"Create repository"**

### Langkah 3 — Upload File
**Cara mudah (tanpa install apapun):**
1. Di halaman repo yang baru dibuat, klik **"uploading an existing file"**
2. Drag & drop file `index.html` dan `README.md`
3. Scroll ke bawah, klik **"Commit changes"**

**Cara via Git (opsional, untuk developer):**
```bash
git init
git add .
git commit -m "first commit: landing page Nikah Yukk"
git branch -M main
git remote add origin https://github.com/USERNAME/nikah-yukk.git
git push -u origin main
```

### Langkah 4 — Aktifkan GitHub Pages
1. Buka tab **Settings** di repository
2. Scroll ke bawah, klik **"Pages"** di menu kiri
3. Di bagian **Source**, pilih **"Deploy from a branch"**
4. Pilih branch **"main"** dan folder **"/ (root)"**
5. Klik **Save**
6. Tunggu 1–2 menit, website akan live di:
   `https://USERNAME.github.io/nikah-yukk`
